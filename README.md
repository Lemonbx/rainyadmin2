### 🌈 技术栈魔法

#### 🍃 后端魔法 (Backend)
- **核心框架**: `Spring Boot 3.5.8` + `Kotlin 1.9.25` ✧*｡
- **数据守护**: `Spring Data JPA` + `Hibernate 6.6` (配合 `QueryDSL 7.1` KSP 版)
- **安全卫士**: `Sa-Token 1.44.0` (轻量级权限认证)
- **工具百宝箱**: `Hutool 5.8.10` + `MapStruct 1.6.3`
- **数据库**: `MySQL`
- **JDK**: `Java 21` (现代化的心跳)

#### ☀️ 前端小太阳 (Frontend)
- **核心框架**: `React 19.2` + `Vite` (由 `rolldown` 驱动) ⚡️
- **路由管理**: `React Router 7.11`
- **状态管理**: `Zustand 5.0.9` (像棉花糖一样轻量)
- **数据交互**: `TanStack Query v5` + `Axios`
- **UI 组件**: `Ant Design 6.1.1` (像彩虹糖一样治愈)
- **构建工具**: `Vite`
